# 🪐 Exoplanet Personality Profiler

> Convert 4,367 rows of NASA numerical exoplanet data into rich natural language  
> descriptions, then apply NLP tasks — all visualised in an interactive space-themed dashboard.

---

## Project Structure

```
exoplanet_project/
│
├── Planets_list.csv          ← NASA Exoplanet Archive dataset (place here)
│
├── data_cleaner.py           ← Step 1: Load, clean & EDA plots
├── text_generator.py         ← Step 2: Template-based description generation
├── nlp_pipeline.py           ← Steps 3-5: Sentiment · Habitability · Clustering
│                                         Outputs planet_data.json + planet_profiles.csv
│
├── index.html                ← Dashboard frontend (loads outputs/planet_data.json)
├── style.css                 ← All styling — dark space theme
├── app.js                    ← All JS: charts, word clouds, search, starfield
│
└── outputs/
    ├── planet_data.json      ← Generated by nlp_pipeline.py (loaded by dashboard)
    ├── planet_profiles.csv   ← Full enriched dataset with all NLP columns
    └── eda/
        ├── 01_discovery_methods.png
        ├── 02_orbital_period_hist.png
        ├── 03_mass_vs_radius.png
        ├── 04_top_observatories.png
        └── 05_controversial.png
```

## Setup

### 1. Install dependencies

```bash
pip install pandas matplotlib scikit-learn
```

### 2. Place the dataset

Copy `Planets_list.csv` into the project root (same folder as the `.py` files).

### 3. Run the NLP pipeline

```bash
# From the project root:
python nlp_pipeline.py
```

This runs all 5 steps in sequence and writes:
- `outputs/planet_data.json`   — aggregated stats for the dashboard
- `outputs/planet_profiles.csv` — full enriched CSV (4,367 rows)

To also generate EDA plots:

```bash
python data_cleaner.py
```

Plots are saved to `outputs/eda/`.

### 4. Serve the dashboard

The dashboard fetches `outputs/planet_data.json` via `fetch()`, so it must be  
served from a local HTTP server (not opened directly as a file).

```bash
# Python built-in server — run from the project root:
python -m http.server 8000
```

Then open **http://localhost:8000** in your browser.

## Pipeline Details

| File | Step | Description |
|------|------|-------------|
| `data_cleaner.py` | 1 | Load CSV, select columns, fill NaNs with group medians, save EDA plots |
| `text_generator.py` | 2 | Template engine → one rich paragraph per planet |
| `nlp_pipeline.py` | 3 | VADER-style sentiment scoring (compound score + label) |
| `nlp_pipeline.py` | 4 | Rule-based habitability scoring (9-point rubric) |
| `nlp_pipeline.py` | 5 | TF-IDF + K-Means clustering into 6 planet archetypes |

## Dashboard Features

| Section | Description |
|---------|-------------|
| **Stat Cards** | Total planets, % positive, % habitable, most promising planet |
| **Sentiment by Method** | Bar chart — which discovery method produces the most positive descriptions |
| **Sentiment Distribution** | Doughnut — Positive / Neutral / Negative counts |
| **Word Clouds** | Top 22 words in Positive vs Negative descriptions |
| **Habitability Pie** | Potentially Habitable / Marginally Interesting / Uninhabitable |
| **Discovery Method Bar** | Planet counts per method |
| **Facility Sentiment Bar** | Top observatories ranked by avg sentiment |
| **Bubble Chart** | Mass vs Orbital Period — bubble size = sentiment, color = habitability |
| **Top 10 Table** | Most promising planets by combined score (40% sentiment + 60% habitability) |
| **Planet Search** | Live search with autocomplete across all 4,367 planets |

## Key NLP Insights

- **80%+ positive** descriptions — habitable-zone hints and Sun-like stars dominate
- **Imaging & Astrometry** score most negative (cool dwarfs, elliptical orbits)  
- **Sentiment correlates with habitability** — confirmed by bubble chart clustering  
- **Kepler-87 c** is the #1 most promising world (hab score 9/9 + positive sentiment)
